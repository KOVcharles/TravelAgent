# Evidence rules

- Prefer a source with file name, page, section, and effective date when available.
- Fall back to the file name when detailed metadata is unavailable.
- Never infer a numeric limit, allowed transport class, approval requirement, or reimbursement rule without retrieved evidence.
- If two sources conflict, mark the item `unknown` and list both sources for administrator review.
- Quote only a short excerpt needed to explain the conclusion.
